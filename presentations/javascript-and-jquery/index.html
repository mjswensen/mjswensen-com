Fundamentals

JavaScript's syntax is C-like.

{% highlight js %} if(hungry) eat(); foo.bar = 'baz'; var time = new Date(); {% endhighlight %}

JavaScript employs function scoping, which is different than block scoping.

{% highlight js %} var myFunc = function() {// New scope here. var myVar = 'Hello, world.'; if(true) {// NO new scope here! var myVar = 'Goodbye.';// This would be a redeclaration of myVar and will produce unexpected results as well as errors/warnings for linters. } }; {% endhighlight %}
