# mjswensen.com

Source code for [mjswensen.com](https://mjswensen.com/).

## Developing locally

```sh
docker-compose up
```

The site will be available on http://localhost:5000.

## Deploying

```sh
git push
```

## Get the date/time for front matter

```sh
date "+%Y-%m-%d %H:%M:%S %Z" | pbcopy
```
